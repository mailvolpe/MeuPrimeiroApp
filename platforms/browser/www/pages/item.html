Conteúdo em detalhes

Você vai utilizar páginas de imagem com texto para s itens de conteúdo do seu aplicativo.

O usário navega pelas categorias do seu app e quando encontra o item desejado, este é o modelo de página ideal para representá-lo

Alguns recursos

Você pode usar outros elementos como:

  • Subtítulos
  • Listas
  • Tabelas
  • Imagens
  • Botões

Funcionalidades

Estas são algumas das funcionalidades que você poderá inserir no seu aplicativo:

Revelar áreas ocultas

Com a funcionalidade de revelar áreas ocultas, você pode criar conteúdo que só aparece quando o usuário aciona os botões corretos. Dessa forma é possível criar várias funcionalidades para o seu aplicativo.

Revelar conteúdo

Esta área só apareceu porque você clicou no botão correto. É possível usar esse recurso para criar diversos tipos de apps.

Navegação externa

É possível utilizar links externos para direcionar o usuários para páginas de vendas, sites, blogs ou qualquer outro conteúdo na internet.

Abrir link externo

Navegação interna

A navegação interna serve para conduzir o usuário através das páginas e conteúdo do seu aplicativo.
